About Nightlight Game Launcher

What is Nightlight Game Launcher?

NGL is a program that helps you launch games without additional logins

(eg. if you play GTAV on Steam and it asks you for a Rockstar Social Club Account and you don't have one, with the help of this program you will skip those logins).

How this launcher works?

NGL works on a simple principle

  • Installs remade game files.
  • Replaces original files with modded ones.
  • Starts the game.

Nightlight Launcher does not support:

  • FiveM.
  • GTAV Online.
  • RDR2 Online
